Phishing URL Detection Tool 🔐
This is a Python-based GUI application that detects suspicious or phishing URLs using basic rule-based analysis.

🚀 Features
Detects non-secure websites (HTTP)
Identifies long and suspicious URLs
Detects special characters like '@' and '-'
Flags phishing-related keywords (login, verify, bank)
🛠️ Technologies Used
Python
Tkinter
💡 How to Use
Enter a URL in the input box
Click on "Check URL"
View the result (Safe or Suspicious)
📌 Example
http://bank-login.com → ⚠️ Suspicious
https://google.com → ✅ Safe
